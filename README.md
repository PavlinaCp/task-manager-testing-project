 # Task Manager - Testovací projekt
 
 Jednoduchý CLI správce úkolů napsaný v Pythonu.
 
 Projekt zaměřen především na tvorbu testovacích scénářů (test cases).
 
 Tento projekt byl vytvořen jako součást zadání v rámci akademie.
 
 Aplikace umožňuje uživateli:
  - přidat úkol
  - zobrazit všechny úkoly
  - odstranit úkol
    
## Testování

Bylo provedeno manuální testování a vytvořena dokumentace ve formě testovacích scénářů.

Testování pokrývá:
- validní vstupy
- nevalidní vstupy
- hraniční vstupy
- práci s více úkoly

Testovací scénáře jsou součástí repozitáře ve formě PDF dokumentu.

##  Spuštění programu
1. Otevřete terminál ve složce projektu.
2. Spustě program příkazem: python main.py

Program se spustí v konzoli a zobrazí hlavní menu správce úkolů.
