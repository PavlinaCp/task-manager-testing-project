# Task Manager - Testing Project
Simple CLI task manager written in Python.
Project mainly focused on creation of test cases.

This project was created as part of an accademy assignment.
This aplication allows users to:
- add a task
- display all tasks
- remove a task

## Testing 
Manual testing was performed and documented using test cases.
Test coverage includes:
- valid inputs
- invalid inputs
- boundary values
- empty state scenarios
- multiple task handling

  Test cases are included in the repository as a PDF document.
